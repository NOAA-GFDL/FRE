versions(1)
===========

NAME
----
versions - A terse cvs status wrapper

SYNOPSIS
--------
[verse]
versions [<args>]

DESCRIPTION
-----------
*versions* is a wrapper to \'cvs status\' that is more terse in its
output.  Mainly prints only filename, and CVS branch/version number,
and the current status of the file.

OPTIONS
-------
_<args>_::
Additional options/directories to pass to 'cvs status'.

SEE ALSO
--------
*cvs*(1)
