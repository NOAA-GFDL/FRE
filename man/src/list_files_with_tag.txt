list_files_with_tag(1)
=====================

NAME
----
list_files_with_tag - Return a list of files in CVS with a given tag name

SYNOPSIS
--------
[verse]
list_files_with_tag <cvs_tag>

DESCRIPTION 
----------- 
*list_files_with_tag* will return a list of files under CVS control,
relative to the current directory, which can be updated or checked out
with the _<cvs_tag>_ tag or branch name.

OPTIONS
-------
_<cvs_tag>_::
A CVS tag or branch name.
